# Dataset Download Instructions

This project utilized the dataset titled 'Indicators of Heart Disease (2022 Update) from Kaggle.

Here are the instructions for downloading the data set.
1. Go to this link which sends you to the Kaggle website: [link](https://www.kaggle.com/datasets/kamilpytlak/personal-key-indicators-of-heart-disease)
2. Download the dataset from Kaggle.
3. Extract `heart_2020_cleaned.csv`.
4. Place the file inside the `data/` folder.
